Варианты заданий

1. Треугольник Серпинского

Этот L-система описывает создание фрактала треугольника Серпинского.

    Алфавит: F, G, +, -
    Аксиома: F-G-G
    Правила:
        F → F-G+F+G-F
        G → GG
    Угол: 120°

Описание: Каждый шаг итерации разбивает треугольник на новые треугольники, создавая фрактал Серпинского.

2. Кривая «Распадающаяся спираль»

Фрактал, напоминающий закручивающуюся спираль, которая разветвляется и распадается на каждом шаге.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F
    Угол: 30°

Описание: Этот фрактал создаёт спиралевидную структуру с ветвями, которые отходят от спирали, разветвляясь наружу и образуя сложный рисунок.

3. Драконий фрактал

Драконий фрактал известен своей извилистой формой, напоминающей путь молнии.

    Алфавит: F, X, Y, +, -
    Аксиома: FX
    Правила:
        X → X+YF+
        Y → -FX-Y
    Угол: 90°

Описание: Символы X и Y используются как промежуточные шаги, не отображаемые на экране, и управляют генерацией фрактала.

4. Дерево Пифагора

Это L-система генерирует фрактальное дерево, в котором каждая ветвь делится на две подветви.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F[+F]F[-F]F
    Угол: 25°

Описание: Ветви дерева растут, и каждая ветвь в процессе роста делится на две подветви. Символы [ и ] используются для того, чтобы управлять вложенностью ветвей.

5. Кривая Леви

Кривая Леви — это самоподобный фрактал, который напоминает форму спирали с закруглёнными углами.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → -F+F-
    Угол: 45°

Описание: Каждый шаг итерации разбивает отрезок на три части, образуя угол в 45°, что постепенно превращает начальный отрезок в извилистую кривую.

6. Кривая Гильберта

Кривая Гильберта — это заполняющая пространство кривая, которая рекурсивно делит квадрат на всё более мелкие сегменты.

    Алфавит: L, R, +, -
    Аксиома: L
    Правила:
        L → +RF-LFL-FR+
        R → -LF+RFR+FL-
    Угол: 90°

Описание: Кривая Гильберта проходит через все точки внутри квадрата, заполняя пространство в виде непрерывной линии.

7. Кривая Пеано

Кривая Пеано также является заполняющей пространство кривой, но имеет более сложную рекурсивную структуру.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F+F-F-F-F+F+F+F-F
    Угол: 90°

Описание: Эта кривая разбивает пространство на квадраты и постепенно их заполняет, образуя узор, похожий на сетку.

8. Треугольник Коха

Этот фрактал похож на кривую Коха, но строится на основе треугольников.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F+F−F−F+F
    Угол: 120°

Описание: Каждая сторона треугольника заменяется на цепь из трёх новых отрезков, формируя самоподобную структуру, которая с каждой итерацией становится всё более сложной.

9. Двойной дракон Хартера-Хейтуэя

Этот вариант драконьего фрактала создает два противоположно направленных фрактала.

    Алфавит: F, X, Y, +, -
    Аксиома: FX+FX-
    Правила:
        X → X+YF+
        Y → -FX-Y
    Угол: 90°

Описание: Двойной дракон создаёт две ветви, которые закручиваются в противоположных направлениях, создавая сложные завитки и формы.

10. Квадратный фрактал (крест)

Этот фрактал напоминает квадратную сетку с дополнительными элементами, выходящими за её пределы.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F+F-F-F+F[F-F-F+F]F
    Угол: 90°

Описание: Фрактал строится из квадрата, к которому добавляются новые «ветви» из других квадратов на каждом шаге, создавая структуру, напоминающую крест с ветвящимися квадратами.

11. Кривая Госкена

Кривая Госкена — это сложная фрактальная кривая, которая с каждой итерацией образует всё большее количество острых углов.

    Алфавит: F, +, -
    Аксиома: F-F-F-F
    Правила:
        F → F-F+F+FF-F-F+F
    Угол: 90°

Описание: На каждой итерации структура разветвляется, создавая узор, похожий на сложный лабиринт, образованный из прямых углов.

12. Фрактал крест

Этот фрактал напоминает крест с множеством ветвей, выходящих из каждого угла на каждом уровне рекурсии.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F
    Угол: 90°

Описание: С каждой итерацией из каждой ветви "креста" выходит новый крест, создавая сложный симметричный узор.

13. Кривая Серпинского (вариант)

Этот вариант кривой Серпинского использует другие правила для формирования треугольного фрактала с более сложной структурой.

    Алфавит: A, B, +, -
    Аксиома: A
    Правила:
        A → B-A-B
        B → A+B+A
    Угол: 60°

Описание: Этот фрактал создаёт треугольную форму с меньшими треугольниками на каждой итерации, формируя самоподобную структуру, похожую на треугольник Серпинского.

14. Фрактальное дерево с углами

Фрактальное дерево, где ветви не только разделяются, но и изгибаются под определёнными углами.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → FF[+F][-F]
    Угол: 30°

Описание: На каждом уровне рекурсии дерево разделяется на две новые ветви, которые затем изгибаются в разные стороны. Результат напоминает дерево с ветвями, изгибающимися на разных углах.

15. Кривая дракона Хартера-Хейтуэя (вариант)

Это другой вариант драконьего фрактала, но с более сложной структурой для каждой ветви.

    Алфавит: F, X, Y, +, -
    Аксиома: X
    Правила:
        X → X+YF+
        Y → -FX-Y
    Угол: 45°

Описание: Этот вариант дракона создаёт более резкие углы и более плотную структуру на каждой итерации, придавая фракталу ощущение "вихря", которое продолжается по мере роста ветвей.

16. Кривая Минковского

Кривая Минковского — это самоподобная кривая, которая рекурсивно разбивает каждый отрезок на более мелкие сегменты, образуя сложную сетчатую структуру.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F+F-F-F+F
    Угол: 90°

Описание: Эта кривая создаёт прямоугольную сетку с каждым новым уровнем рекурсии, формируя узор, который можно использовать для моделирования фрактальных структур.

17. Фрактал Хевея

Этот фрактал похож на кривую дракона, но с другим распределением ветвей, создавая более сложную структуру на каждом уровне.

    Алфавит: F, X, +, -
    Аксиома: F
    Правила:
        F → F+F-F
        X → FX-FX
    Угол: 120°

Описание: Фрактал Хевея создаёт сложную структуру ветвления, похожую на дерево, но с несимметричными ветвями и углами поворота, что даёт уникальный визуальный эффект.

18. Кривая Леви (вариант)

Этот вариант кривой Леви изменяет базовое правило для более сложного роста фрактала с крутыми изгибами.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → +F--F+
    Угол: 45°

Описание: С каждым шагом эта кривая образует множество уголков, которые напоминают самоподобные спирали и зигзагообразные линии.

19. Кривая «Ковёр Серпинского»

Этот фрактал создаёт двухмерную фрактальную структуру, напоминающую ковёр, который состоит из всё меньших квадратов на каждой итерации.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F+F-F-F[+F-F]+F
    Угол: 90°

Описание: Этот фрактал создаёт узор в виде квадрата, в котором добавляются новые квадраты на каждом уровне, постепенно формируя структуру, похожую на ковёр.

20. Кривая Леви с петлями

В этой версии кривой Леви добавляются элементы, которые создают петли, формируя более сложный узор.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F]F[-F]F
    Угол: 45°

Описание: Этот фрактал на каждой итерации создаёт петли и дополнительные ответвления, которые образуют сложную и детализированную самоподобную структуру.

21. Фрактал Кантера

Этот фрактал использует деление отрезков на несколько частей, убирая центральную часть, что приводит к созданию множества небольших сегментов.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F+F--F+F
    Угол: 90°

Описание: На каждой итерации отрезок делится на несколько частей, что создаёт фрактальный узор с пустотами. Это вариант одномерного ковра Серпинского.

22. Кривая Антуана

Кривая Антуана — это фрактал, который имеет самоподобную структуру с множеством небольших петель и изгибов.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F+F-F+F
    Угол: 90°

Описание: На каждом уровне эта кривая образует множество петель и углов, что приводит к созданию сложного фрактального узора.

23. Квадратный фрактал (ковёр)

Этот фрактал создаёт ковровую структуру, похожую на ковёр Серпинского, но с использованием квадратных элементов.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[-F][+F]F
    Угол: 90°

Описание: С каждым шагом на месте каждого квадрата появляются новые квадраты, которые создают ковровую фрактальную структуру.

24. Кривая Линденмайера

Эта кривая создаётся с помощью сложных правил разветвления, образуя ветвистую структуру.

    Алфавит: F, X, +, -, [, ]
    Аксиома: X
    Правила:
        X → F[+X][-X]FX
        F → FF
    Угол: 25°

Описание: Кривая Линденмайера создаёт деревообразную структуру с ветвями, которые продолжают разделяться и расти на каждом шаге.

25. Фрактал звездчатого полигона

Этот фрактал создаёт звездообразные структуры, которые появляются на каждом шаге итерации.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F
    Угол: 72° (звезда)

Описание: Эта система создаёт звёздчатые фигуры с дополнительными ветвями, которые добавляются на каждом шаге итерации. Угол в 72° создаёт форму пятиконечной звезды.

26. Кривая террактального дерева

Это фрактальное дерево отличается от других своей сложностью и множеством ответвлений на каждом уровне итерации.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F[+F][-F]
    Угол: 25°

Описание: На каждом шаге добавляются новые ветви, которые наклоняются в разные стороны. Ветвистая структура дерева становится всё более сложной с каждой итерацией.

27. Кривая Кочехеда

Этот фрактал использует механизм ветвления, создавая множество маленьких отрезков, которые формируют сложную и запутанную фигуру.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → F+F−F−F+F
    Угол: 90°

Описание: Этот фрактал напоминает сетку с множеством ветвей, которые разделяются на каждом уровне и образуют сложный узор.

28. Снежинка Пентагон

Этот фрактал основан на пятиугольной структуре, которая разветвляется на каждом уровне итерации, создавая фигуру, похожую на снежинку.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][++F][-F][--F]
    Угол: 72°

Описание: Фрактал формирует пятиугольные структуры с ветвями, которые образуют звездообразную фигуру, похожую на снежинку.

29. Кривая Гексагональной звезды

Этот фрактал создаёт звездообразную структуру, исходящую из шестиугольника.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F
    Угол: 60°

Описание: Эта система генерирует шестиугольные формы с ответвлениями, которые разветвляются и создают звездообразную структуру на каждом шаге.

30. Фрактал «Кривая Змея»

Фрактал, который напоминает извивающуюся змею с поворотами и ветвлениями на каждом шаге.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]FF
    Угол: 30°

Описание: Эта кривая извивается и образует сложные узоры, которые напоминают движения змеи. Ветвления добавляют дополнительную сложность структуре.

31. Кривая Пифагора

Этот фрактал создаёт древовидную структуру с прямыми углами, напоминающую классическое "Дерево Пифагора".

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F
    Угол: 45°

Описание: На каждом уровне появляются новые ветви под прямыми углами, создавая густую древовидную структуру. Ветви продолжают разветвляться в геометрической прогрессии.

32. Кривая Т-образного дерева

Этот фрактал создаёт дерево с Т-образными разветвлениями на каждом уровне.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F]F[-F]F
    Угол: 90°

Описание: Эта структура создаёт ветви, которые всегда делятся на две новые ветви, расположенные под прямыми углами, создавая Т-образные структуры на каждом шаге.

33. Кривая Л-снежинка

Этот фрактал представляет собой вариацию снежинки, но с добавлением больше углов и ответвлений.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[-F][+F]FF
    Угол: 60°

Описание: Эта кривая создает шестиугольные и звездообразные структуры, напоминая снежинки с более сложными ветвями и угловатыми элементами.

34. Кривая «Кольцо»

Фрактал, создающий кольцеобразную структуру, с ветвями, которые исходят из центра и развиваются наружу.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F
    Угол: 120°

Описание: На каждом шаге образуются ветви, которые выходят из центральной точки и образуют кольцеобразные структуры. С каждым уровнем количество ветвей увеличивается, заполняя окружность.

35. Кривая Коха с ветвями

Этот вариант кривой Коха использует ответвления, создавая дополнительные структуры на каждом шаге.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F+F--F[+F][-F]F
    Угол: 60°

Описание: Фрактал создаёт форму, похожую на кривую Коха, но с добавлением ветвлений на каждом шаге. Это даёт более сложную и интересную структуру, которая напоминает одновременно и кривую, и древовидную структуру.

36. Фрактал «Кристалл»

Этот фрактал напоминает кристаллическую структуру, разветвляясь с каждым шагом на несколько ветвей.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F]F[-F]F
    Угол: 60°

Описание: Каждая итерация добавляет новые ветви, формирующие многослойную кристаллическую структуру. Угол в 60° создаёт шестигранную симметрию, что придаёт форму снежинки или кристалла.

37. Кривая Аполлона

Этот фрактал основан на круговой симметрии, где каждый сегмент делится на два с поворотом.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F[+F][-F]
    Угол: 45°

Описание: Эта структура образует фрактальный узор, похожий на крионку Аполлона, с множеством симметричных ветвей, расходящихся в разные стороны.

38. Фрактал «Лист»

Этот фрактал напоминает форму листа с разветвляющимися прожилками.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]
    Угол: 30°

Описание: Каждый уровень итерации создаёт структуру, напоминающую лист дерева, с основным стеблем и отходящими веточками, которые становятся всё мельче.

39. Кривая Хиберта

Этот фрактал заполняет двумерное пространство, создавая замкнутые петли и изгибы, которые проходят через все возможные области.

    Алфавит: A, B, +, -
    Аксиома: A
    Правила:
        A → +BF-AFA-FB+
        B → -AF+BFB+FA-
    Угол: 90°

Описание: Кривая Хиберта создаёт структуру, которая постепенно заполняет всю плоскость, используя четкие повороты на 90°. Это классический фрактал для заполнения пространства.

40. Фрактал «Звезда с ветвями»

Этот фрактал формирует звездообразную структуру с ответвлениями на концах каждой звезды.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F[+F][-F]
    Угол: 72°

Описание: Этот фрактал создаёт многолучевую звезду, на каждом из лучей которой появляются новые ветви. Симметрия углов в 72° формирует пятиугольную звезду с фрактальными ответвлениями.

41. Фрактал «Паук»

Фрактал с симметричным ветвлением, напоминающим паучьи лапки, отходящие от центрального тела.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F][-F]F[+F][-F]F
    Угол: 45°

Описание: Этот фрактал создаёт симметричную структуру, похожую на паучьи лапки, с каждым уровнем добавляя новые сегменты в разные направления.

42. Кривая Леви

Этот фрактал образует угловатую, «рваную» линию, которая с каждым шагом становится всё более сложной.

    Алфавит: F, +, -
    Аксиома: F
    Правила:
        F → +F--F+
    Угол: 45°

Описание: Кривая Леви создаёт замысловатую форму, которая с каждым шагом строится из ломаных линий и острых углов. Это классическая кривая для изучения фрактальных линий.

43. Фрактал «Цветок»

Этот фрактал напоминает лепестки цветка, которые появляются на каждом шаге итерации.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F]F[-F]F
    Угол: 45°

Описание: Каждый новый шаг создаёт ветви, которые отходят от центрального стебля под углом, формируя структуру, похожую на распускающийся цветок.

44. Кривая Пентафрактала

Фрактал на основе пятиугольника с ветвлениями, выходящими от каждой стороны.

    Алфавит: F, +, -, [, ]
    Аксиома: F
    Правила:
        F → F[+F]F[-F]F[+F]F
    Угол: 72°

Описание: Каждый шаг итерации приводит к образованию пятиугольных структур с разветвлениями, создавая узор, похожий на пятиконечную звезду или пентагон с ветвями.
